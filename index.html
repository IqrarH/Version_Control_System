Version Control Using GitHub

In this blog we'll discuss about Version Control System GitHub. We'll learn how to create a repository on GitHub. How to initialize your local repository to connect to the remote repository on GitHub. How to to commit and push your code to the remote repository. What are conflicts and how to avoid and resolve those conflicts.

First let's create a remote repository on GitHub. A repository can be either public or private. Public Repository is visible to everyone on GitHub, whereas private repository is visible only to you or the collaborators you add to that repository.
